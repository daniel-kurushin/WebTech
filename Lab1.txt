# Для выполнения работы была выбрана страница сайта elecomp.ru с поиском (elecomp.ru/search)
# Программные инструменты: "инструменты разработчика" в браузере Firefox и консольная программа wget.
# С помощью браузера был рассмотрен запрос, отсылаемый сайту для поиска нужной информации.
# Было установлено, что сайту передается всего один параметр: searchstring.
# Затем в консоли windows был выполнен следующий скрипт:

cd "c:\users\Хаким"
wget --post-data=searchstring=test11111111111 https://elecomp.ru/search/

# Данным скриптом был произведен поиск на указанном сайте и создан файл index.html с результатом поиска.
